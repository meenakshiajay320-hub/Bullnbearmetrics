# Stock Market Volatility Analysis

## Overview
This project analyzes stock market data using Python to study price trends and volatility.  
It retrieves historical stock data and applies statistical techniques to understand market behavior.

## Objectives
• Analyze historical stock prices  
• Calculate simple prediction based on 52WH,52WL and piotroski scores
• Visualize price trends  

## Technologies Used
- Python
- Pandas
- Matplotlib

## Key Concepts Demonstrated
- Financial data analysis
- Data visualization
